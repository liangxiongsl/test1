mini_magick——图像处理

MiniMagick 是封装了 ImageMagick 图像处理库 

# MiniMagick 命令  
magick

使用 mini_magick 之前需要安装 MiniMagick 

require 'mini_magick'  
  
image = MiniMagick::Image.open("input.jpg")  
  
# 调整图像尺寸  
image.resize "200x200"  
  
# 调整图像质量  
image.quality(80)  
  
# 裁剪图像  
image.crop "100x100+50+50"  
  
# 添加滤镜  
image.combine_options do |c|  
  c.modulate "120,150,100"  
  c.gamma     1.2  
  c.contrast  
end  
  
# 输出  
image.write "output.jpg"

github 上传，读取，删除（图像）文件

post的关键有三个：PATH（url 中的后面的部分），HEADER，BODY

参考文档 

require 'open-uri'  
require 'net/http'  
require 'json'  
require 'base64'  
  
# 实现 github 官方给出的 api 接口  
def public_send(method,file_path:'',git_path:nil,body:{},msg:nil)  
  git_path=git_path||"test/#{File.basename(file_path)}"  
  url=URI("https://api.github.com/repos/liangxiongsl/image-repo/contents/#{git_path}")  
  
  case method  
  when 'get'  
    req=Net::HTTP::Get.new(url)  
    req['Authorization']='token ...'  
  when 'put'  
    req=Net::HTTP::Put.new(url)  
    req['Authorization']='token ...'  
    req.body={  
      'message':msg||'upload file',  
      'content':Base64.encode64(File.binread(file_path))  
    }.to_json  
  when 'delete'  
    req=Net::HTTP::Delete.new(url)  
    req['Authorization']='token ...'  
    req.body={  
      'message':msg||'delete file',  
      'sha':JSON.parse(public_send('get',file_path:file_path,git_path:git_path))['sha']  
    }.to_json  
  end  
  
  Net::HTTP.start(url.hostname,url.port,use_ssl: true ) do |http|  
    http.request(req)  
  end.body  
end  
  
# 1)上传  
ret=public_send('put',file_path:'cxk.jpg')  
puts ret  
  
# 2)读取  
ret=public_send('get',file_path:'cxk.jpg')  
puts ret  
File.binwrite('cxk_get.jpg',Base64.decode64(JSON.parse(ret)['content']))  
  
# 3)删除  
ret=public_send('delete',file_path:'cxk.jpg')  
puts ret

批量重命名，移动文件

1. 遍历目录/文件夹
2. 文件批量重命名或移动（如：删除某个子串） 

require 'find'  
require 'fileutils'  
Find.find('E:\LX\_武装库\华章数学') do |filename|  
  begin  
    File.rename(filename,filename.gsub("华章数学译丛",""))  
    # FileUtils.mv(filename,"/new")  
  rescue  
  end  
end  
  
# 注：filename 属于字符串，内容为"路径+文件名"  
# 注2：文件重命名后，filename将无效

"完整路径" -> "祖先路径" 或 "文件名" 或 "扩展名"

require 'File'  
  
path = "/path/to/xyz.mp4"  
  
File.basename(path)         # => "xyz.mp4"  
File.basename(path, ".mp4") # => "xyz"  
File.basename(path, ".*")   # => "xyz"  
File.extname(path)          # => ".mp4"  
File.dirname(path)          # => "/path/to"

创建，读取，修改压缩包 zip

gem 'rubyzip'  
require 'zip'  
# 创建 **.zip 文件, 若文件不存在  
Zip::File.open("test.zip",Zip::File::CREATE) do |zip|  
  # 在 **.zip 内创建条目(文件)  
  zip.get_output_stream("tt.txt") do |f|  
    f.puts "here is a txt file in tt.txt"  
    f.puts "writing something here..."  
  end  
end  
  
Zip::File.open("test.zip",Zip::File::CREATE) do |zip|  
  zip.each do |f|  
    # f 为条目名(文件名)  
    puts f  
    # 输出文件内容  
    puts zip.read(f)  
  end  
end

socket——客户端，服务端通信？

require 'socket'  
serv=TCPServer.open('localhost',4000)  
loop do  
  client=serv.accept  
  # 返回给客户端时间  
  client.puts(Time.now.ctime)  
  # 返回给客户端消息  
  client.puts("hello client!")  
  client.close  
end
