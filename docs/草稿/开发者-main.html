DevOps

——使开发者和运维人员参与进来，携手合作，改进软件交付

简化版

  • linux => 文件系统，包管理器，管理服务，检查日志，bash，权限，管道，输出重定向，文本操作的常用工具，进程监控，CLI 编辑器等
  • 网络和协议 => DNS, TCP/IP, SSH, 端口，网关，路由，IP 寻址，子网划分
  • Docker => 容器化；熟系 Alpine Linux
  • git => DevOps 团队经常练习 git ops
  • AWS => 创建虚拟应用并部署到 AWS
  • Terraform => 自动创建基础结构
  • Ansible => 配置管理；编写自动化脚本，如 .db 备份
  • github actions => CI/CD 工具；将 CI/CD 集成到应用程序
  • Nginx => 用于 web 服务，反向代理，负载均衡，媒体流等
