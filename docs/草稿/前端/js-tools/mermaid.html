cdn 安装 => 选择一个合适的 cdn 版本
最新版 =>

  • fastly.jsdelivr.net/npm/mermaid@10.9.0/dist/mermaid.min.js
  • fastly.jsdelivr.net/npm/mermaid@10.9.0/dist/mermaid.esm.min.mjs
    => 两种引入方式：
<script src="https://fastly.jsdelivr.net/npm/mermaid@10.9.0/dist/mermaid.min.js"></script>

<script type="module">
	import mermaid from 'https://fastly.jsdelivr.net/npm/mermaid@10.9.0/dist/mermaid.esm.min.mjs';
	mermaid.initialize({ startOnLoad: true });
</script>

node 安装 => npm i mermaid

mermaid-cli => npm install -g @mermaid-js/mermaid-cli
（需要设置环境变量 PUPPETEER_SKIP_DOWNLOAD 为 chrom.exe 的路径！）

注1：

左值 右值 含义 别名

}o o{ 0或多个 1. many(0)

|| || 1 1. only one
2. 1
}| |{ 至少1个 1. one or more/many
2. many(1)
3. 1+

注2：左右值的别名：

左值/右值 含义 注释
0或1 one和zero可互换

-- 或 to => 直接关系（以直线表示）
.. 或 optionally to => 间接关系（以虚线表示）
