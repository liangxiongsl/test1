ajax

XMLHttpRequest（XHR）用于解决网页更新内容的问题，而无需重新加载网页

Ajax（Asynchronous JavaScript and XML）技术就包含了 XMLHttpRequest api 以实现浏览器和服务器之间传递数据，而无需重新加载网页

通常，Ajax 不能跨域工作。例如，从 example1.com 加载的网页无法向 example2.com 发出 Ajax 请求，因为它会违反同源策略

跨域资源共享（CORS） => 允许 ajax 请求到不同的域名

ajax()
get()

关键概念

ajax 通常需要指定获取数据的类型的指令，有以下几种选择：

数据类型 解释
text 传输简单字符串
html 传输要防止在页面上的html块
script 向页面添加新脚本
json 传输 json 格式字符串（格式不正确可能导致请求失败，而不报错）
jsonp 跨域传输 json
xml 用于在自定义 XML 架构中传输数据

ajax 是异步的

基本方法

$.ajax(config: object)

config 的选项：

  • url => 请求 url
  • type => 请求方法类型
  • data => 请求携带的数据
  • dataType => 期望返回的数据

.done( (data: any) => any ) => 请求成功时的回调
.fail( (xhr: object, st: string, errThrown) => any ) => 请求失败时的回调
.always( (xhr: object, st: string) => any ) => 无论请求是否成功都执行的回调

其他：async, cache, context, jsonp, timeout, traditional

例子：

$.ajax({
    url: 'https://www.travisblog.asia/search',
    type: 'get',
    data: {keywords: '蔡徐坤'},
    dataType: "json"
}).done((data)=>{
    console.log(data)
}).fail((xmr, st, err)=>{

}).always((xmr, st)=>{

})

ajax 与表单

序列化

表单数据由一系列键值对构成，而表单中 select / checkbox / radio 等元素的 name 属性对应 ajax 序列化后的 字符串 或 对象数组的 key

<form>.serialize(): string => 将表单数据序列化为查询字符串，如：select-checked=example-val&radio-checked=on&checkbox-checked=on
<form>.serializeArray(): object[] => 将表单数据序列化为对象数组，如：

[
	{name: 'selecte-checked', value: 'example-val'},
	{name: 'radio-checked', value: 'on'},
	{name: 'checkbox-checked', value: 'on'}
]
客户端验证

参考：animate.style

原理 => 可以侦听提交事件 submit，然后使用 <event>.preventDefault() 阻止表单的发送（期间可以进行一些错误提示）

例子：（这里使用了 animate.css 提供的特效）

// 表单验证失败时调用的特效
function animate(x){
    x.css('background-color', 'red')
    x.css('color', 'red')

	// x[0] 可以得到 dom 元素
    x[0].classList.add('animate__animated', 'animate__headShake')
    x[0].addEventListener('animationend', ()=>{
        x.removeClass('animate__animated')
        x.removeClass('animate__headShake')
        x.css('background-color', '')
        x.css('color', '')

    }, {once: true})
}

const form = $j('#form')
const name = $j('#name')
const password = $j('#password')
$j( "#form" ).submit((e)=>{

    if (name.val().length === 0){
        animate(name)

        e.preventDefault()
    }
    if (password.val().length === 0){
        animate(password)

        e.preventDefault()
    }
})
预过滤器

预过滤器 $.ajaxPrefilter() 在每次请求之间可以预处理请求的选项对象 options

$.ajaxPrefilter() 检测跨域 ajax 请求，并转发给代理的例子

$j.ajaxPrefilter((options, originalOptions, jqXHR) => {
    if (options.crossDomain){
        options.url = "http://mydomain.net/proxy/" + encodeURIComponent(options.url)
        options.crossDomain = false
    }
})
