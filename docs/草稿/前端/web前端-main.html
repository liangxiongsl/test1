block element modifier

block__element--modifier

  • block => 块，即某种 ui 区域
  • element => 某种块下的子元素，可以有多个层级关系
  • modifier => 该元素的修饰符
    例子：.body, .body--theme-dark；.form, .form__submit, .form__submit--disabled

渲染技术

外链：CSR，SSR和SSG是什么，有什么优缺点？ - 掘金 (juejin.cn), 新一代Web建站技术栈的演进：SSR、SSG、ISR、DPR都在做什么？ - 知乎 (zhihu.com)

概念 译名 解释 优点 缺点
CSR(Client Side Rendering) 客户端渲染 渲染工作在客户端进行（如： vue, react） 1. 前后端分离 => 前端专注于界面开发，后端专注于 api 开发
2. 减轻服务器压力		 1. 不利于 SEO => 搜索引擎不执行 js 操作
2. 首屏渲染较慢
SSR(Server Side Rendering) 服务端渲染 渲染工作在服务端进行（如：jsp, php） 1. 有利于 SEO
2. 有利于首屏渲染		 1. 用户体验差 => 每次跳转页面需要在服务端重新渲染
2. 服务器压力大
SSG(Static Site Generation) 静态网站构建 开发完成后构建结果页面，每次访问时直接返回静态 html 1. 有利于 SEO
2. 减轻服务器压力		 1. 只适用于静态数据，每次改动需要重新构建
2. 用户体验差 => 每次打开新页面需要重新渲染
ISR(Incremental Site Rendering) 增量式网站构建
DPR(Distributed Persistent Rendering) 分布式持久渲染

问题

a 标签的 href

  • https://example.com => 绝对url
  • /path/to => 相对 url（绝对 path） => 容易被 Servlet 或 Spring MVC 解析
  • path/to => 相对 url（相对 path）
  • #section => 锚链接
  • mailto:info@example.com => 邮箱
  • <a href="javascript:void(0);" onclick="myFunction()">Click me</a> => 调用 js 函数（js 代码）
