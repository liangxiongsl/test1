手动安装 aehd（用 sdk manager 会安装失败）

  • 安装release => google/android-emulator-hypervisor-driver (github.com)
  • （可选）将解压目录放于 <android-sdk>/extras/google/aehd-windows 下
  • 运行 aehd-windows 目录下的 silent_install.bat（管理员身份）
  • 检测安装是否成功 => <android-sdk>/emulator/emulator.exe -accel-check
    注：CPU必须有虚拟化扩展，并且 BIOS 未禁用该扩展
    注2：必须禁用 Hyper-V

    • 有线：设备启用开发者选项和USB 调试
    • 无线：在硬件设备上运行应用  |  Android Studio  |  Android Developers (google.cn)

安装依赖 => flutter pub add http

例子：

import 'package:http/http.dart' as http;
http.get(Uri.parse('https://jsonplaceholder.typicode.com/albums/1'))
  .then((value) => {
    print(value.body)
  });

例子2：

import 'package:http/http.dart' as http;
import 'dart:convert';
http.post(Uri.parse('http://10.0.2.2:8000/'), headers: {
      'Content-Type': 'application/json; charset=UTF-8',
    }, body: jsonEncode({
      'data': {
        'hello': 'world'
      }
    })).then((value) => print(value.body));

例子3：

import 'package:http/http.dart' as http;
import 'dart:convert';
class MyHttp{
  String base_url = 'http://10.0.2.2:8000';
  MyHttp();
  MyHttp.base(this.base_url);

  Future<http.Response> get(String path) async{
    return await http.get(Uri.parse('${base_url}/${path}'), headers: {
      'Content-Type': 'application/json; charset=UTF-8',
    });
  }
  Future<http.Response> post(String path, Map body) async{
    return await http.post(Uri.parse('${base_url}/${path}'), headers: {
      'Content-Type': 'application/json; charset=UTF-8',
    }, body: jsonEncode(body)
    );
  }
}
持久化

三种持久化方式

  • shared_preferences
    • 以 key-val 的方式存储基本数据类型（int, double, bool, String, List<String>）
    • 不能确保重启应用后数据仍然存在
  • 文件
  • SQlite

shared_preferences => flutter pub add shared_preferences
=> 例子： 

import 'package:shared_preferences/shared_preferences.dart';
SharedPreferences.getInstance().then((sp){
  sp.setString('key', 'value');
  print(sp.getString('key'));  
  sp.remove('key');
});

注：安装依赖后 app 需要重新编译

文件方式存储数据 => flutter pub add path_provider
=> 例子：

import 'package:path_provider/path_provider.dart';
import 'dart:io';
getApplicationDocumentsDirectory().then((dir){
  print(dir);
  File f = File('${dir.path}/test.txt');
  f.writeAsStringSync('hello world');
  print(f.readAsStringSync());
});

注：android 用户一般只能浏览 /storage/emulated/0/ 目录
注2：android 中，path_provider 可以返回目录 /data/user/0/<example.app.package>/app_flutter, /data/user/0/<example.app.package>/files, /data/user/0/<example.app.package>/cache

例子2：（无状态组件）

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context){
    return const MaterialApp(
      home: Scaffold(
        body: Center(
          child: Text('hello world'),
        ),

    );
  }

例子3：（有状态组件）

import 'package:flutter/material.dart';
import 'package:http/http.dart' as http;
void main() => runApp( MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context){
    return const MaterialApp(
      home: MyWidget()
    );
  }
}

class MyWidget extends StatefulWidget{
  const MyWidget({super.key});

  @override
  State<StatefulWidget> createState() => MyState();
}
class MyState extends State<MyWidget>{
  var res = '';
  
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text('hello world'),
            ElevatedButton(child: Text('press me'), onPressed: () {
              http.get(Uri.parse('http://exmaple.com'))
                  .then((value) => res = value.body)
                  .then((value) => setState(() {}));
            }),
            Text(res),
          ],
        ),
      ),
    );
  }
}
