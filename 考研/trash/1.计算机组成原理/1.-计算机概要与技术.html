计算应用的分类及其特性

关键字：个人计算机(personal computer, PC)，服务器(server)，嵌入式计算机(embedded computer)，超级计算机，处理器核

计算机主要包括三类：

计算机类型 描述 特点
个人计算机 用于个人使用的计算机，通常包含图形显示器、键盘和鼠标等 针对单用户提供良好的性能，价格低廉，通常运行第三方软件
服务器（大型机） 用于为多用户运行大型程序的计算机，通常由多个用户并行使用，并且一般通过网络访问 有更强的计算、存储、I/O能力；但故障恢复代价高得多，更强调可靠性
嵌入式计算机 嵌入到其他设备的计算机，一般运行预定义的一个或一组应用程序 数量最多，应用广泛；广泛使用处理器核，参见4. 处理器
超级计算机 高端服务器，拥有最高性能和最高成本

程序概念入门

关键字：应用软件，系统软件(system software)，硬件，操作系统(operating system)，服务，监控，编译程序(complier)，翻译，解释，二进制位(binary digit)，位(bit)，指令(instruction)，数据(data)，汇编程序(assembler)，汇编语言(assembly language)，助记符，机器语言(machine language)，高级编程语言(high-level programming language)，编译程序，MIPS

应用程序的层次结构由外到内依次为 => 应用软件，系统软件，硬件

系统软件 => 提供常用服务的软件，包括操作系统、编译程序、加载程序、汇编程序等

(1) 操作系统 => 为了使程序更好地在计算机上运行而管理计算机资源的监控程序；是用户程序和硬件之间的接口，为用户提供各种服务和监控功能

  • 处理基本的输入和输出操作
  • 分配外存和内存
  • 为多个应用程序提供共享计算机资源的服务
    => 例子：Linux、IOS、Windows

(2) 编译程序 => 将高级语言编写的程序翻译为计算机所能识别的机器语言的指令

(3) 汇编程序 => 将指令由助记符形式翻译为二进制形式

汇编语言 => 以助记符表示的机器指令
机器语言 => 以二进制元形式表示的机器指令

二进制位（位）：基数为 2 的一位数字 0 或 1；信息的基本组成单元
指令（狭义）：由二进制数据构成的，计算机硬件能理解并服从的命令

语言 指令形式（广义） 转换程序 转换行为
高级语言 程序/代码 编译程序 翻译/解释
汇编语言 助记符(机器指令) 汇编程序 汇编

显示器

关键字：图形显示器，液晶显示(Liquid Crystal Display, LCD)，液体聚合物，动态矩阵显示(active matrix display)，晶体管，像素(pixel)，像素矩阵，位图(bit map)，三原色，光栅，帧缓冲区，刷新频率
关键字引用：PMD

液晶显示 => 一种显示技术，用液体聚合物薄层的带电或不带电来传输或阻止光线的的传输
大多数 PMD 的图形显示器使用液晶显示 LCD，其显示效果是轻巧、低功耗

动态矩阵显示 => 一种液晶显示技术，使用晶体管控制单个像素上光线的传输
像素 => 图像元素的最小单元，屏幕由成千上万的像素组成的矩阵而形成
大多数 LCD 的显示器采用动态矩阵显示技术，其每个像素由一个晶体管精确地控制电流，使图像更清晰

图像由像素矩阵组成，可以表示成二进制位的矩阵，称为位图

彩色显示器 => 使用 8 位表示每个三原色（红、绿，蓝），每个像素用 24 位表示

计算机硬件采用光栅刷新帧缓冲区来保存位图以支持图像

触摸屏

关键字：触摸屏，电容感应

PC 使用 LCD，而后 PC 时代的 PMD 使用接触敏感的显示设备——触摸屏——来替代 键盘、鼠标

触摸屏 => 可采用多种方式实现，多数采用电容感应实现
