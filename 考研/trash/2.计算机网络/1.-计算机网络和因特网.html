什么是因特网

什么是因特网？这个问题有两种描述方式：

  1. 从因特网的具体构成——基本硬件，软件组件——来描述
  2. 从为分布式应用提供服务的联网基础设施来描述

具体构成描述

关键字：计算机网络(computer network)，主机(host)，端系统(end system)，通信链路(communication link)，分组交换机(packet switch)，同轴电缆，铜线，光纤，无限定频谱，传输速率(transmission rate)，bps(bit/s)，分组(packet)，路由器(router)，链路层交换机(link-layer switch)，路径(route/path)，分组交换网络，因特网服务提供商(Internet Service Provider, ISP)，wifi接入ISP，蜂窝数据ISP，住宅宽带接入，高速局域网接入，移动无线接入，内容提供者，较低层ISP，较高层ISP，协议(protocol)，传输控制协议（Transmission Control Protocol, TCP），网际协议（Internet Protocol），TCP/IP，因特网标准(Internet standard)，因特网工程任务组(Internet Engineering Task Force, IETF)，请求评论(Request For Comment, RFC)，IEEE

因特网的一些部件

通信链路 => 类型：同轴电缆，铜线，光纤，无线电频谱
=> 传输速率：bps(bit/s)

2 个端系统的数据通信的例子：
[端系统]将数据分组（即数据分段，每段加上手段字节） => [网络]数据传输 => [端系统2]装配数据

分组交换机：
=> 原理：从入通信链路接受到达的分组 => 选择一条出通信链路转发分组
=> 分类：路由器，链路层交换机

  • 路由器用于网络核心中
  • 链路层交换机用于接入网中

路径：分组从发送端系统到接收端系统，所经历的一系列通信链路和分组交换机

分组交换网络

因特网服务提供商 ISP
=> 本地电缆，住宅区ISP，公司ISP，大学ISP，机场/咖啡店/公共场所提供 wifi 接入的 ISP，智能手机等提供移动接入的 蜂窝数据 ISP
=> 提供的网络接入：住宅宽带接入（线缆调制解调器/DSL），高速局域网接入，移动无线接入
=> 较低层 ISP 通过较高层 ISP（Level 3 Communications、AT&T、NTT）互联起来
=> 较低层 ISP 通过高速光纤链路互联的高速路由器组成

协议——协议控制因特网中信息的接受和发送
=> 最重要的两个协议：TCP, IP

  • IP：定义了在路由器和端系统之间发送和接收的分组格式
    => 因特网主要协议的统称 => TCP/IP

因特网标准由因特网工程任务组 IETF 研发
=> 请求评论 RFC 为 IETF 的标准文档

IEEE 组织
=> IEEE 802 LAN/MAN 标准化委员会指定了以太网和无线 wifi的标准 => 针对网络链路的标准

服务描述

关键字：分布式应用程序(distibuted application)，基础设施，套接字接口(socket interface)

分布式应用程序 => 涉及多个相互交换数据的端系统的应用程序 => 如：多人游戏，视频会议，在线社交论坛，位置共享服务

基础设施 => 为应用程序提供服务

套接字接口 => 规定了运行在端系统上的程序请求因特网基础设施向运行在另一个端系统上的特定目的地程序交付数据的方式
如：A 向给 B 发送邮箱，A就必须提供寄信标题、寄信内容、收件人地址等内容数据

什么是协议

协议

人类活动的类比：

网络协议 => 定义了两个或多个通信实体之间交换的报文的格式和顺序，以及报文发送和/或接受一条报文或其他事件所采取的动作

因特网上，涉及两个或多个远程通信实体的所有活动都收协议制约，例如：

对象/场景 协议 作用
两台物理上连接的计算机 硬件实现的协议 控制了在两块网络接口卡间的“线上”的比特流
端系统 拥塞控制协议 控制了发送方和接收方之间传输的分组发送的速率
分组传输 路由器中的协议 决定了分组从源到目的地的路径
